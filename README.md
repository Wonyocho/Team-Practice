## [과제] 웹페이지 만들기.
디렉토리 구조를 참고하여, 정적 웹페이지를 html과 css로 만들어 보세요.
파트는 다음과 같습니다.

- 성준형
    - 회사 소개 페이지를 구현하세요.
    - 회사 채용공고 페이지를 구현하세요.
	- 모든 페이지는 '반응형 레이아웃'을 적용해야 합니다.
	- 각 페이지에는 이미지가 반드시 들어가야 합니다.

- 최준호:
  - 웹 환경에서 작동하는 간단한 게임을 구현하세요.
	- 난이도 선택이 가능해야 합니다. (이지, 노말, 하드)
	- CSS 디자인이 적용되어야 합니다.
	- 점수를 로컬 스토리지에 저장 가능해야 합니다.
